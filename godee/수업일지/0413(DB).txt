Ch04 그룹함수 사용하기 

SUM : 그룹의 누적 합계를 반환합니다. 
AVG : 그룹의 평균을 반환합니다. 

*
COUNT :그룹의 총 개수를 반환합니다. 



MAX: 그룹의 최대값을 반환합니다. 

MIN : 그룹의 최소값을 반환합니다. 

->그룹함수 중 MIN, MAX함수는 숫자데이터 이외의 다른 모든 데이터 유형에 사용할 수 있습니다. 

----------------
로우 개수를 구하는 COUNT 함수 

테이블의 행 개수를 반환합니다. 

COUNT( {* |  [DISTINCT|ALL] expr }  )

COUNT( *  ) 은 NULL을 포함한 모든 행의 개수를 구합니다. 



---------------

데이터 그룹: GROUP BY

->그룹함수는  GROUP BY 절 뒤에 어떤 컬럼값을 기준으로 그룹함수를 적용해 줄지 기술해 주어야 하는데 해당 컬럼을 기술합니다. 
이때  GROUP BY 절 다음에는 *컴럼의 별칭을 사용할 수 없고 반드시 컬럼명을 기술해야 한다는 점에 주의해야합니다.* 


--------------
그룹 결과 제한: HAVING 절

GROUP BY
HAVING

->HAVING 절을 사용하여 표시할 그룹을 지정하여 집계 정보를 기준으로 그룹 결과를 제한할 수 있습니다.
