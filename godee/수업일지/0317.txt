<form> 태그

<form> 태그는 입력을 위한 HTML 양식을 작성하는데 사용됩니다. 

*action 속성 : 양식을 제출할 때 양식 데이터를 보낼 위치를 지정합니다. (입력데이터의 전달 위치를 지정합니다.)

*method 속성 : 양식 데이터를 보낼 때 사용할 HTTP 메서드를 지정합니다. (입력 데이터의 전달 방식을 지정합니다.)

name 속성 : 양식의 이름을 지정합니다. 

enctype 속성 : 서버에 양식 데이터를 제출할 때 양식 데이터를 인코딩하는 방법을 지정합니다. 


----------------------------------

<fieldset>태그

<fieldset> 태그는 양식에서 관련 요소를 그룹화 하는데 사용합니다. 

disabled 속성 : 관련 양식 요소 그룹을 사용하지 않도록 지정합니다. 

form 속성 : <fieldset>가 속하는 양식을 지정합니다. 

name 속성 : <fieldset>의 이름을 지정합니다. 

-----------------------------------

<legend> 태그

<legend> 태그는 <fieldset> 요소의 캡션을 정의합니다. 

---------------------
<input> 태그

<input> 태그는 사용자가 데이터를 입력 할 수 있는 입력 필드를 지정합니다. 

<input> 요소는 <form> 요소 내에서 사용되어 사용자가 데이터를 입력할 수 있는 입력 컨트롤을 선업합니다. 

입력 필드는 type 속성에 따라 여러가지 방법으로 달라질 수 있습니다. 

placeholder 속성 : <input> 요소의 예상 값을 설명하는 짧은 힌트를 지정합니다. 

readonly 속성 : 입력 필드가 읽기 전용임을 지정합니다. 

*type 속성 : 표시할 <input> 유형을 지정합니다. 
------------
다양한 input 태그

<input type ="tel">은 전화번호를 입력하기 위한 필드를 정의합니다.

<input type="password">는 비밀번호 필드를 정의 합니다. (문자가 마스크 됩니다.)

<input type="text">는 한 줄 텍스트 필드를 정의합니다.
	-> 택스트 필드의 기본 너비는 20자입니다. 

<input type="submit">은 모든 양식 값을 양식 핸드러에 제출 하는 제출 버튼을 정의합니다. 
	-> 양식 핸들러는 일반적으로 입력 데이터를 처리하기 위한 스크립트가 있는 서버 페이지 입니다. 양식 핸들러는 양식의 action 속성에 지정합니다. 

<input type="reset">은 모든 양식 값을 초기 값으로 재설정하는 재설정 버튼을 정의합니다. 

<input type="image">는 이미지를 제출 버튼으로 정의합니다. 
	-> 이미지의 경로는 src 속성에 지정합니다.

<input type="checkbox">는 확인란을 정의합니다.
	확인란은 활성화 될 때 체크 표시되는 사각형 상자로 표시됩니다. 
	확인란은 사용자가 제한된 수의 선택 항목 중 하나 이상을 선택할 수 있도록 하는데 사용합니다.(체크박스는 중복, 라디오 박스 단일)

<input type="radio">는 라디오 버튼을 정의합니다. 
	라디오 버튼은 일반적으로 라디오 그룹(관련 옵션 세트를 설명하는 라디오 버튼 모음)으로 표시합니다. 
	한 그룹에서 하나의 라디오 버튼만 동시에 선택할 수 있습니다. (동일한 그룹안에 있어야함)

<input type ="hidden">은 숨겨진 입력 필드를 정의합니다 
	숨겨진 필드를 통해 웹개발자는 양식을 제출할 때 사용자가 보거나 수정할 수 없는 데이터를 포함할 수 있습니다. 
	숨겨진 필드는 종종 양식을 제출할 때 업데이트 해야 할 데이터 베이스 레코드를 저장합니다. 
	
<input type="number">는 숫자를 입력하기 위한 필드를 정의합니다. 
	max : 허용되는 최대값을 지정합니다. 
	min : 허용되는 최소값을 지정합니다. 
	step : 유효한 숫자 간격을 지정합니다. 
	value : 지정된 기본값

<input type ="file">은 파일 선택 필드와 파일 업로드를 위한 "찾아보기" 버튼을 정의합니다.

<label> 태그

<label> 태그는 여러 <input> 요소에 대한 레이블을 정의합니다. 

<label> 태그는 for 속성은 관련 <input> 요소의 id 속성과 같아야 합니다. 

------------------

<select> 태그

<select> 요소는 드롭다운 목록을 작성하는 데 사용됩니다. 

<option>의 <select> 요소 내부 태그 목록에서 사용할 수 있는 옵션을 정의합니다. 

size 속성 : 드롭 다운 목록에 표시되는 옵션의 수를 정의합니다. 

name 속성 : 드롭 다운 리스트의 이름을 정의합니다. 

multiple 속성 : 한번에 여러 옵션을 선택할 수 있도록 지정

-----------------

<option> 태그 

<option> 태그는 논리(boolean) 속성입니다. 

존재하는 경우 페이지가 로드될 때 옵션을 미리 선택하도록 지정합니다. 

미리 선택된 옵션이 드롭 다운 목록에서 먼저 표시됩니다. 

selected 속성 : 기본값으로 설정할 속성

-----------------
<optgroup> 태그

<optgroup> 태그는 드롭다운 목록에서 관련 옵션을 그룹화하는데 사용됩니다. 

옵션 목록이 길면 관련 옵션 그룹이 사용자들 보다 쉽게 처리할 수 있습니다. 

label 속성 : 옵션 그룹의 레이블을 지정합니다. 

------------------

<textarea> 태그

<textarea> 태그는 여러줄로 된 텍스트 입력 컨트롤을 정의합니다. 
텍스트 영역에는 무제한의 문자가 포함될 수 있으며, 텍스트는 고정폭 글꼴로 렌더링 됩니다. 

텍스트 영역의 크기는 cols 및 rows 속성으로 지정할 수 있습니다. css의 width, height 속성을 통해 더 좋게 지정할 수 있습니다. 

*cols속성 : 텍스트 영역의 보이는 너비를 지정합니다. 

*rows속성 : 텍스트 영역에 표시할 수 있는 줄 수를 지정합니다. 

name 속성 : 텍스트 영역의 이름을 지정합니다. 

maxlength 속성 : 텍스트 영역에 허용되는 최대 문자 수를 지정합니다. 

-----------------

<button>태그

<iframe> 태그

<iframe> 태그는 인라인 프레임을 지정합니다. 

인라임 프레임은 현재 HTML 문서 내에 다른 문서를 포함시키는데 사용합니다. 

name 속성 : 요소의 이름을 지정합니다. 

scrolling 속성 : 스크롤 바를 표시할지 여부를 지정합니다 .
(yes,no,auto)

width 속성 : 요소의 너비를 지정합니다. 기본 너비는 300px입니다. 

height 속성 : 요소의 높이를 지정합니다. 기본높이는 150px입니다. 

src : 속성 요소에 포함할 문서의 주소를 지정합니다. 

-------------
get 방식

get은 지정된 리소스에서 데이터를 요청하는데 사용됩니다. 

get은 가장 일반적인 http방법중 하나 입니다. 

쿼리 문자열(name/value의 쌍)은 get 요청의 URL로 전송됩니다. 

	ex>test.jsp?name1=value1&name2=value2

get 요청은 캐시 할 수 있습니다. 

get 요청은 브라우저 기록에 남아있습니다. 

get 요청을 북마크 할 수 있습니다. 

민감한 데이터를 다룰 때에는 get 요청을 사용해서는 안됩니다. 

get 요청에는 길이 제한이 있습니다. 

get요청은 데이터를 요청하는 데만 사용됩니다. 

--------------
post 방식

post는 리소스를 생성, 업데이트 하기 위해 서버로 데이터를 전송하는 데 사용됩니다. 

post와 함께 서버로 전송된 데이터는 http요청의 요청 본문에 저장됩니다. 

post는 가장 일반적인 HTTP 방법 중 하나입니다. 

post 요청은 캐시되지 않습니다. 

post 요청은 브라우저 기록에 남아 있지 않습니다. 

post 요청은 북마크 할 수 없습니다. 

post 요청에는 데이터 길이에 대한 제한이 없습니다. 

--------------

<link>태그

<link>태그는 문서와 외부 리소스 간의 링크를 정의합니다. 

<link> 태그는 외부 스타일 시트에 연결하는 데 사용 됩니다. 

href 속성 : 링크된 문서의 위치를 지정합니다. 

rel 속성 : 현재 문서와 링크된 문서 간의 관계를 지정합니다.*(필수)

type 속성 : 링크된 문서의 미디어 유형을 지정합니다.

<script> 태그

<script> 태그는 클라이언트 측 스크립트(javascript)를 정의하는 데 사용 됩니다. 

<script> 요소는 스크립팅 명령문을 포함하거나 src 속성을 통해 외부 스크립트 파일을 가리킵니다. 

javascript의 일반적인 용도는 이미지 조작, 양식 유효성 검사 및 동적 컨텐츠 변경입니다. 

type 속성 : 스크립트의 미디어 유형을 정합니다. 

src 속성 : 외부 스크립트 파일의 URL을 지정합니다.

<span> 태그는 텍스트 또는 문서의 일부를 추가하는 방법을 제공합니다.
